Tutorial 1

Getting Started with FlashPhoto


Introduction

FlashPhoto is a digital painting program that also allows for powerful photo-editing. The application itself can be thought of as two applications in one. FlashPhoto's functionality is divided into painting functions and filter functions.


Brushes

When you first open up FlashPhoto, you will be greeted with a blank canvas. This canvas can be drawn on with a variety of included tools. With FlashPhoto, you are able to select one of these tools along with the current tool's color. If you click and drag your mouse within the boundaries of the canvas window, your selected tool will be applied to the canvas at the location indicated by your mouse cursor. This will leave a trail of virtual paint on the canvas. If you wish to correct any mistakes, you can use the provided 'eraser tool'. This tool acts in the same way as others, but removes any paint instead of applying it. 'Undo' and 'Redo' options have also been provided. The following is a complete list of tools available in FlashPhoto:

  • Pen

  • Eraser

  • Spray Can

  • Calligraphy Pen

  • Highlighter

  • Chalk

  • Stamp

  • Blur

You can select your tool of choice under 'Tool Type' in the top left corner of the control screen.





Filters

One of the more unique features of FlashPhoto is the ability to apply different filters to an image. FlashPhoto supports uploading both JPEG and PNG image files to the canvas. By selecting an image, you will have the option to click the 'Load Canvas' button . When an image is loaded to the canvas, you can think of it as your new canvas background. You will be able to draw on it just as you would with a blank canvas. In addition to loading images, FlashPhoto allows you to save the changes you have made to the uploaded image. This can be done by selecting the 'Save Canvas' button. See the image below for a visual description.









Once an image is loaded, you may apply one of many integrated filters to it. The following is a list of included filters and each of their intended effects:

  • Blur

    • Blurs the image proportional to the specified amount.

  • MotionBlur

    • Blurs the image in a directional manner. Uses the amount specified as the distance of the effect in each direction.

  • Sharpen

    • Sharpens the image proportional to the specified amount.

  • Edge Detect

    • Detects and highlights the edges of objects in an image.

  • Threshold

    • Each color channel is rounded up to the maximum value of 1.0 or down to the minimum value of 0.0 based on whether the pixel's value is greater or less than the specified input.

  • Saturation

    • Increases, decreases, or inverts the color in an image.

  • Channels

    • Increase or decrease the intensity of each color channel.

  • Quantize

    • Reduces the number of possible colors in an image by binning each color value into the number of specified bins. For example, if using 4 bins, there will be 4 possible intensity values for each color channel, spaced evenly: 0%, 33%, 66%, 100%.
