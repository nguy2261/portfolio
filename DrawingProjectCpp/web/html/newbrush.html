Adding a New Brush

One of the main features of FlashPhoto is its brushes, or tools. We have implemented our own tools by using a ToolFactory. Using a ToolFactory makes the process of adding a new tool very simple.


Step 1

The first step in creating your own tool is to create the tool's C++ and header files. Because both FlashPhoto and Mia use tools, all tools should be placed in the /libphoto/src directory. As far as naming convention goes, we name our tool files as follows: 'T<tool function>.cpp, T<tool function>.h'. Here are some example C++ and header files from our Pen tool. Our Pen tool uses our provided 'MConstant' mask. Keep in mind that you may need to provide your own mask, depending on the tool you will be creating.

















Step 2

Secondly, we will want to add a reference to our tool's header file in /libphoto/include/libphoto.h. Because your tool files will be in a separate directory, please add the reference as follows: '#include “../src/T<tool name>.h”' Below is an illustration of our current libphoto.h file.



















Step 3

Lastly, you will want to add your new tool to our ToolFactoryFlash C++ and header files, which can be found in the /FlashPhoto/src directory. Simply add your new tool to the list of tools in the factory. See visual description below.
