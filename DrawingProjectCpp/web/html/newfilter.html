Adding a New Filter

Adding a new filter to FlashPhoto is just as easy as adding your own brush. We have implemented our own filters by using a FilterFactory. Using a FilterFactory makes the process of adding a new filter very simple. All filters have their own C++ and header files. Some filters use a regular filter (Filter.h), while others use a convolution filter (FilterConvolve.h). Some examples of convolution filters we have provided are our Blur filter, Edge Detection filter, MotionBlur filter, and Sharpen filter. Each of these convolution filters also have kernel files. Kernel files should be named as follows: 'K<filter function>.cpp, K<filter function>.h'.


Step 1

The first step in creating your own filter is to create the filter's C++ and header files. Because both FlashPhoto and Mia use filters, all filters should be placed in the /libphoto/src directory. As far as naming convention goes, we name our filter files as follows: 'F<filter function>.cpp, F<filter function>.h'. Here are some example C++ and header files from our Edge Detection filter. Step 2

Secondly, we will want to add a reference to our filter's header file in /libphoto/include/libphoto.h. Because your filer files will be in a separate directory, please add the reference as follows: '#include “../src/F<filter function>.h”'.
