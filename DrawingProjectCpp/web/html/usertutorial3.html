Converting and Segmenting a CT Scan using Mia


One useful feature that Mia offers is the ability to convert and segment CT scans. This process will take advantage of Mia's ability to apply filters to a 'stack' of images in a directory. In order to complete this process properly, you will need a directory full of multiple images in the following format: 'image###.png'. If set up properly, your directory of images should look similar to the following:














The result of this process will leave us with a new image stack, but the images will be quantized and will have 4 tissue types.


Step 1

The first step is to slightly blur all of the images in the directory. By doing so, we reduce noise in the images.


Step 2

The second step is to quantize the resulting images into 4 segments. The result will show 4 tissue types.


This process can be completed with a single call in command line mode:

'./Mia inDir/image###.png -blur <float> -quantize 4 outDir/image###.png'


The resulting images will be saved into a new directory. We have provided a directory of sample brain scan images for you to experiment with. This directory can be found at '/Mia/brainstack'.
