Getting Started with Mia


Introduction

Mia is a Medical Imaging Application. You can think of Mia as a streamlined and stripped down version of FlashPhoto, with features essential to medical professionals. Mia offers two essential tools for annotating images; a red pen, and a red 'Arrow' stamp used to mark areas on images. The following filters have been carried over from FlashPhoto:

  • Sharpen

  • Edge Detect

  • Threshold

  • Multiply RGB (named 'Channels' in FlashPhoto)

  • Quantize

In addition to these filters, Mia provides a Grayscale tool that will make your image appear black and white.


Loading and Annotating Images

Images can be loaded with the same method used in FlashPhoto. Simply select an image (JPEG or PNG), and select the 'Load Canvas' button. Mia also allows you to load a 'stack' of images. This can be done by providing Mia a directory of images with the naming convention imageName###.png or imageName###.jpg. By loading the first image (for example, imageName000.png) you will be able to use the 'Next Image' and 'Previous Image' buttons to quickly page through your stack of images. Image annotation can be performed in the same manner as in FlashPhoto. Simply choose a tool, and click on the canvas to apply it. However, Mia also offers a more streamlined method for editing medical photos. This method is referred to as 'Command Line Mode'. With command line mode, you can simply run Mia with an additional command option. The following is a list of all command line arguments:


  • -h

  • -sharpen <integer>

  • -edgedetect

  • -thresh <float>

  • -quantize <integer>

  • -blur <float>

  • -saturate <float>

  • -multrgb <float> <float> <float>

  • -compare


The following is a list of example applications of the command line arguments (executing './Mia -h' will show you these same examples, along with the full list above).


  • Convert an image in JPG format to PNG format:

    • ./Mia in.jpg out.png


  • Convert an image in PNG format to JPG format:

    • ./Mia in.png out.jpg


  • Apply sharpening filter:

    • ./Mia in.png -sharpen <int> out.png


  • Apply edge detection filter:

    • ./Mia in.png -edgedetect out.png


  • Apply threshold filter:

    • ./Mia in.png -thresh <float> out.png


  • Apply quantize filter:

    • ./Mia in.png -quantize <integer> out.png


  • Apply blur filter:

    • ./Mia in.png -blur <float> out.png


  • Apply saturation filter:

    • ./Mia in.png -saturate <float> out.png


  • Apply multiply RGB filter:

    • ./Mia in.png -multrgb <float> <float> <float> out.png


  • Image comparison:

    • ./Mia image1.png -compare image2.png


  • Apply a command to each image in a directory:

    • ./Mia inDir -<argument> outDir


  • Apply multiple filters to an image (this example shows two filter operations, but Mia supports an arbitrary number of filter operations that are specified in the order they should be applied):

    • ./Mia in.png -sharpen 5 -edgedetect out.png
